# todo
To-Do List Web App

A dynamic to-do list web application. Allows users to manage tasks, create lists, and organize items

    Add and manage multiple to-do lists.
    Add, edit, and delete tasks within each list.
    Drag and drop tasks to reorder them (optional).
    Mark tasks as completed and clear completed tasks.
    Responsive design for mobile and desktop use.
    Persistent storage using LocalStorage (or backend database if required).

Technologies Used

    Frontend:
        HTML5
        CSS3 (Bootstrap 5.3)
        JavaScript (ES6+)

    Version Control:
        Git

    Optional Features:
        External APIs for data (if required)
        Backend integration (e.g., Node.js, Express, MongoDB)
